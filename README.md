# Gose - A terascale file-uploader

## Features

- Upload progress-bar
- Direct upload to Amazon S3 via presigned-URLs
- Integrity checks via MD5 / Sha256 hashing
- Multi-part upload (planned)
- Resumable multi-part uploads (planned)
- Link shortening (planned)
